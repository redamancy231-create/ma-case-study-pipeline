**重整与和解的程序差异**（建议在第2章以表格呈现）：

| 比较维度 | 重整（第7章，第70-94条） | 和解（第8章，第95-106条） |
| :--- | :--- | :--- |

| 前提条件 | 不能清偿到期债务或明显缺乏清偿能力 | 债权人已申请破产清算、法院受理后、破产宣告前 |

## 第2部分：会计准则与财务分析合规性检查（权重：30%）

### 2.1 企业会计准则适用性核验

| 准则 | 核验内容 | Phase 1处理 | 修正/补充建议 |
| :--- | :--- | :--- | :--- |
| CAS 20 企业合并 | 同一控制（权益结合法/账面价值法）vs 非同一控制（购买法/公允价值法） | ✓已区分 | **重大修正**：美的-小天鹅分类有误（详见下文） |
| CAS 8 资产减值 | 商誉减值测试：年度强制测试+触发测试 | 部分提及 | 补充减值测试方法（可收回金额=公允价值-处置费用与使用价值孰高）、测试单元为资产组或资产组组合 |
| **CAS 33 合并财务报表** | 合并范围确定（控制三要素）、收购NCI的权益性交易处理 | ✗未引用 | **必须补充**：吸收合并后合并范围变化；收购少数股权属权益性交易，不影响损益 |

**关键修正——美的-小天鹅案例的会计分类**：

Phase 1将"美的集团吸收合并小天鹅"分类为**"非同一控制下企业合并，虽为关联方但会计上按购买法处理"**，此分类有误，理由如下：

1. **同一控制认定**：美的集团在合并前已持有小天鹅约37.55%的A股与41.49%的B股股份，为控股股东。双方最终控制人均为何享健，符合CAS 20第5条"参与合并的企业在合并前后均受同一方最终控制且该控制并非暂时性"的定义（控制已持续10年以上）。

2. **更根本的问题**：由于美的集团合并前已控制小天鹅，此次吸收合并实质上是收购少数股东权益（NCI）。根据CAS 33，母公司已控制子公司前提下收购NCI属于**权益性交易**，不确认商誉；对价与NCI账面价值的差额调整资本公积，不影响当期损益。

3. **对报告结构的影响**：Phase 1的案例比较基于"两个同一控制+一个非同一控制"框架以对比会计处理差异。若三个案例均为同一控制，则缺乏非同一控制维度的对比，会计分析深度大打折扣。

**修正建议（推荐方案B）**：

| 方案 | 内容 | 优劣 |
| :--- | :--- | :--- |
| 方案A | 修正美的-小天鹅分类为同一控制/NCI收购，如实分析 | 简单但丧失非同一控制对比维度 |
| **方案B（推荐）** | 替换案例二为**美的集团收购库卡(KUKA)**（2016-2017） | 真正的非同一控制+跨境并购+大额商誉（约22亿欧元）+后续减值讨论充分 |

若采用方案B，报告案例结构变为：

| 案例 | 会计分类 | 产业特征 | 商誉问题 |
| :--- | :--- | :--- | :--- |
| 南车-北车 | 同一控制/权益结合法 | 央企整合/高铁出海 | 无商誉 |
| 美的-库卡 | 非同一控制/购买法 | 民企跨境并购/智能制造 | 大额商誉+减值 |
| 宝钢-武钢 | 同一控制/权益结合法 | 央企整合/供给侧改革 | 无商誉（但资产减值） |

覆盖维度：央企vs民企、同一控制vs非同一控制、国内vs跨境、无商誉vs有商誉——远优于原方案。

### 2.2 财务分析指标体系

Phase 1提出的指标（ROE、ROA、EPS、资产负债率、CAR）覆盖不足，建议扩展为5维度体系：

| 维度 | 指标 | 计算公式 | 并购分析意义 | 正常值参考 | 数据来源 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **盈利能力** | ROE | 净利润/平均净资产 | 并购后股东回报是否提升 | 制造业5%-15% | CSMAR/Wind |
| | ROA | 净利润/平均总资产 | 规模扩张是否伴随效率下降 | 制造业3%-8% | CSMAR/Wind |
| | 销售净利率 | 净利润/营业收入 | 成本协同对利润率的影响 | 行业差异大 | 年报 |
| | EBITDA margin | EBITDA/营业收入 | 剔除折旧政策差异后的盈利能力，适合重资产行业 | 行业差异大 | Wind |
| **偿债能力** | 资产负债率 | 总负债/总资产 | 并购融资对资本结构的影响 | 40%-70% | CSMAR/Wind |
| | 流动比率 | 流动资产/流动负债 | 并购后短期偿债能力 | >1.5为安全 | 年报 |
| | 利息保障倍数 | EBIT/利息费用 | 杠杆收购后偿付利息能力 | >3为安全 | 年报 |
| **营运能力** | 总资产周转率 | 营业收入/平均总资产 | 规模扩张后运营效率 | 行业差异大 | CSMAR |
| | 应收账款周转率 | 营业收入/平均应收账款 | 并购后信用政策一致性 | 行业差异大 | 年报 |
| **成长能力** | 营业收入增长率 | (本期-上期)/上期 | 协同效应是否转化为收入增长 | 行业差异大 | CSMAR/Wind |
| | 净利润增长率 | (本期-上期)/上期 | 利润增长可持续性 | 行业差异大 | CSMAR/Wind |
| **市场表现** | EPS | 净利润/加权平均股数 | 换股合并后股本摊薄效应 | — | 年报 |
| | P/E | 每股市价/每股收益 | 市场对并购后盈利的定价 | 行业差异大 | Wind |
| | CAR | Σ(实际收益率-预期收益率) | 市场对并购事件的即时评价 | — | CSMAR/RESSET |

**方法性修正**：
- Phase 1将"毛利率"列入盈利能力维度，建议替换为EBITDA margin：三个案例涉及重资产制造业，EBITDA能更好剔除折旧政策差异
- CAR计算应补充：估计窗口[-120, -31]，事件窗口[-10, +10]，并报告[-5,+5]、[-10,+10]、[-30,+30]的稳健性检验结果

## 审核结论

核心修正意见共3项（按严重程度排序）：

1. **美的-小天鹅会计分类有误**（严重）：该案例应属同一控制下企业合并/NCI收购，而非Phase 1所载"非同一控制下企业合并"。此错误导致报告丧失非同一控制分析维度。建议替换为美的-库卡案例以恢复比较框架。

2. **破产法条文范围严重遗漏**（严重）：Phase 1仅引用第70-81条，遗漏第82-94条（重整计划制定与执行）及第95-106条（和解程序），且未对比中国第75条与美国§362自动中止的差异。

3. **证券法具体条款完全缺失**（重要）：上市公司并购重组必须遵守证券法收购规则，Phase 1未引用任何证券法条款，第2章制度法规部分存在空白。

其余为补充性建议，旨在提升报告的专业性与完整性。以上修正意见供Phase 3撰写与Phase 4交叉验证使用。
