## 2. 分析框架设计

### 2.1 贯穿全文的逻辑分析框架

采用 **"概念界定 → 制度比较 → 方法分类 → 案例实证 → 财务分析 → 结论启示"** 的六步递进框架：

```
概念界定（第2章前半）
 ↓ 回答"是什么"
制度比较（第2章后半）
 ↓ 回答"制度允许什么、约束什么"
方法分类（第3章）
 ↓ 回答"怎么做、会计上如何处理"
案例实证（第4章）
 ↓ 回答"实践中如何运作"
财务分析（第5章）
 ↓ 回答"是否创造了价值、如何体现"
结论启示（第6章）
 ↓ 回答"对理论与实践的启示"
```

| 环节 | 核心论点 | 论据支撑方式 |
| :--- | :--- | :--- |
| **概念界定** | 企业重组的会计本质是对控制权的重新配置，狭义重组强调法律主体变更，广义重组涵盖权利义务的重新安排。 | 引用《企业会计准则第20号——企业合并》对"企业合并"的定义；区分会计、法律、监管三重语境。 |
| **制度比较** | 中国重组制度以行政监管为主（证监会核准、国资委审批），美国以司法主导（Chapter 11 自动中止条款），制度差异塑造了不同的重组路径选择。 | 对比《上市公司重大资产重组管理办法》与Chapter 11的核心程序；引用具体法条。 |
| **方法分类** | 不同重组方法对应不同的会计处理逻辑：同一控制下采用账面价值法，非同一控制下采用购买法，直接影响商誉确认与后续利润波动。 | 依据《企业会计准则第20号》《第12号》；以方法-会计处理对照表呈现。 |
| **案例实证** | 成功案例的共性在于：重组动因与国家战略（高铁出海、供给侧改革、消费升级）契合，且财务整合（采购、研发、渠道协同）在合并后2-3年内显现。 | 三个案例的重组公告、年报、券商研报中的协同效应量化数据。 |
| **财务分析** | 并购重组在短期提升规模指标（营收、总资产），但盈利指标（ROE、ROA）的改善具有滞后性，且商誉减值风险是威胁长期绩效的关键变量。 | 并购前后3年财务指标趋势图、CAR事件研究、商誉占总资产比例变化。 |
| **结论启示** | 会计处理方法是价值创造的"记录器"而非"驱动器"，真正的成功源于战略协同与治理整合；准则制定者应关注商誉后续计量的顺周期效应。 | 综合前文实证结果；引用准则制定机构的讨论稿（如IASB 2013年讨论稿）。 |

### 2.2 数据分析维度

#### 维度一：并购前后盈利能力与运营效率指标变化

| 指标类别 | 具体指标 | 计算方式 | 数据来源 | 分析逻辑 |
| :--- | :--- | :--- | :--- | :--- |
| 盈利能力 | ROE、ROA、毛利率、净利率 | 净利润/净资产、净利润/总资产等 | CSMAR、Wind、上市公司年报 | 对比重组前1年（T-1）、重组当年（T）、重组后2年（T+1、T+2）的趋势，判断盈利改善是否持续。 |
| 运营效率 | 总资产周转率、存货周转率 | 营业收入/总资产平均余额 | CSMAR、年报 | 检验规模扩张是否伴随效率下降（常见陷阱）。 |
| 偿债能力 | 资产负债率、流动比率 | 总负债/总资产、流动资产/流动负债 | CSMAR、年报 | 判断重组是否优化了资本结构。 |

#### 维度二：市场反应与股东价值创造（事件研究法）

| 指标 | 定义 | 计算方式 | 数据来源 | 分析逻辑 |
| :--- | :--- | :--- | :--- | :--- |
| 累计超额收益率 (CAR) | 重组事件窗口期内，股票实际收益率减去预期收益率的累计值 | 市场模型估计预期收益：R_i = α + βR_m + ε；AR = R_i实际 − R_i预期；CAR = ΣAR | CSMAR、RESSET数据库 | 选择事件窗口为重组公告日前后10日（[-10, +10]），检验市场对重组事件的即时评价；区分短期CAR与长期BHAR（买入持有超额收益）。 |
| 股价波动率变化 | 重组前后股票日收益率标准差 | σ = √(Σ(R_i − R̄)² / (n−1)) | CSMAR | 若CAR为正且波动率下降，表明市场视重组为风险降低的价值创造事件。 |

## 3. 案例候选推荐

### 案例一：中国南车吸收合并中国北车（2014–2015）

| 项目 | 内容 |
| :--- | :--- |
| **案例名称** | 中国南车股份有限公司吸收合并中国北车股份有限公司 |
| **重组类型** | 吸收合并（同一控制下企业合并） |
| **选择理由** | ① **政策标杆性**：响应"一带一路"与高铁"走出去"战略，是央企整合的典范；② **会计典型性**：同一控制下合并采用账面价值法，不产生商誉，避免了后续减值风险，会计处理干净；③ **数据可得性**：双方均为A+H股上市公司，信息披露极为充分，合并前后财务数据完整；④ **研究充分性**：已有大量学术文献研究，便于参考文献引用与结论对照。 |
| **关键数据可获取性** | **财务数据**：CSMAR、Wind、中国中车2013–2017年年报；**股价数据**：CSMAR、RESSET；**重组方案**：上交所公告《中国南车股份有限公司与中国北车股份有限公司合并报告书》（2015年1月）。 |
| **预期财务影响方向** | ① 营收与总资产规模倍增（1+1=2的规模效应）；② 由于采用账面价值法，合并后ROE短期可能因净资产增加而被稀释，但协同效应（采购、研发、海外订单统一谈判）在T+2年后逐步释放；③ 毛利率有望提升（消除同业竞争、统一技术标准）；④ 无商誉确认，长期利润表不受减值困扰。 |

### 案例二：美的集团吸收合并小天鹅A（2018–2019）

| 项目 | 内容 |
| :--- | :--- |
| **案例名称** | 美的集团股份有限公司吸收合并无锡小天鹅股份有限公司 |
| **重组类型** | 吸收合并（非同一控制下企业合并，虽为关联方但会计上按购买法处理） |
| **选择理由** | ① **市场化并购代表**：区别于行政主导的央企合并，本案例是民营企业集团内部资源整合，市场化程度高；② **商誉问题典型**：虽为吸收合并，但涉及少数股东权益收购与对价分摊，商誉确认与后续减值测试具有会计学讨论价值；③ **产业逻辑清晰**：白色家电业务互补（小天鹅在洗衣机领域强势），协同效应易于量化；④ **时间较近**：2019年完成，能反映最新会计准则下的处理。 |
| **关键数据可获取性** | **财务数据**：CSMAR、Wind、美的集团2016–2022年年报；**重组方案**：深交所公告《美的集团股份有限公司发行A股股份换股吸收合并无锡小天鹅股份有限公司报告书》（2018年10月）；**券商研报**：中金公司、中信证券关于本次合并的跟踪研报。 |
| **预期财务影响方向** | ① 合并后美的集团洗衣机业务市占率提升，营收结构优化；② 若存在商誉确认，需关注后续年度减值测试对净利润的潜在冲击；③ 整合小天鹅的研发与渠道资源，销售费用率有望下降；④ 由于是换股合并，股本扩张可能短期摊薄EPS，长期看协同效应弥补。 |

### 案例三：宝钢股份吸收合并武钢股份（2016–2017）

| 项目 | 内容 |
| :--- | :--- |
| **案例名称** | 宝山钢铁股份有限公司吸收合并武汉钢铁股份有限公司 |
| **重组类型** | 吸收合并（同一控制下企业合并，双方同受宝武集团控制） |
| **选择理由** | ① **供给侧改革标杆**：2016年国务院发布《关于钢铁行业化解过剩产能实现脱困发展的意见》，本案例是政策落地的标志性事件；② **困境反转属性**：武钢股份在被合并前已连续亏损（2015年巨额亏损），重组带有救助与优化双重性质，财务分析维度丰富；③ **会计处理复杂性**：涉及亏损企业的合并报表处理、资产负债的重新估值；④ **社会影响力大**：涉及大量职工安置与产能淘汰，可拓展讨论重组的社会成本与会计边界。 |
| **关键数据可获取性** | **财务数据**：CSMAR、Wind、中国宝武/宝钢股份2014–2019年年报；**重组方案**：上交所公告《宝山钢铁股份有限公司换股吸收合并武汉钢铁股份有限公司暨关联交易报告书》（2016年9月）；**政策文件**：国务院〔2016〕6号文。 |
| **预期财务影响方向** | ① 去产能后行业供需改善，合并后毛利率回升；② 武钢资产的注入可能短期拉低整体ROE，但通过淘汰落后产能、统一采购（铁矿石议价权提升），T+2年后ROE修复；③ 资产负债率：武钢负债率较高，合并后集团层面统一融资可能优化整体资本结构；④ 无商誉（同一控制），但需关注被合并方资产的减值准备计提是否充分。 |

### 案例四（备选）：顺丰控股借壳上市鼎泰新材（2016）

| 项目 | 内容 |
| :--- | :--- |
| **案例名称** | 顺丰控股借壳马鞍山鼎泰稀土新材料股份有限公司上市 |
| **重组类型** | 借壳上市（重大资产置换 + 发行股份购买资产） |
| **选择理由** | 若正文篇幅允许，可增加此案例以覆盖"借壳上市"这一重要重组类型。顺丰为快递行业龙头，借壳上市时估值与对价安排极具会计学讨论价值（资产置换的公允价值计量、借壳方的业绩承诺与商誉）。但因其重组性质与前三个吸收合并差异较大，若篇幅受限可仅作为第3章"方法分类"中的补充例证，不入第4章主案例。 |
| **关键数据可获取性** | 鼎泰新材/顺丰控股2015–2018年公告、年报；深交所重组问询函及回复。 |
| **预期财务影响方向** | 借壳后上市公司主营业务完全变更，财务指标发生"断层"式变化，不适合做并购前后趋势对比，更适合做"重组日公允价值计量"与"业绩承诺实现情况"的会计分析。 |

**最终建议**：第4章主案例采用**案例一、二、三**（三个吸收合并），覆盖央企行政整合（南车北车、宝钢武钢）与市场化民企整合（美的小天鹅）两种模式，且均为吸收合并，便于横向比较会计处理与财务后果。顺丰借壳上市案例可在第3章"借壳上市"小节中作为简要示例。

## 4. 论述逻辑链设计

### 4.1 完整论述路径

```
起点：企业重组是什么？（概念）
 ↓
制度环境：中国与国际法规如何规范重组？（制度供给）
 ↓
方法工具箱：会计上如何确认、计量、报告？（技术路径）
 ↓
实践验证：典型案例如何运用上述方法？（案例实证）
 ↓
价值检验：财务数据与市场数据是否支持"成功"？（效果评估）
 ↓
终点：对会计准则、监管制度、企业实践的启示。（理论升华）
```

### 4.2 制度法规与案例实证的衔接设计

| 理论铺垫（第2–3章） | 案例应用（第4章） | 衔接逻辑 |
| :--- | :--- | :--- |
| 《公司法》第172条：公司合并需经股东会特别决议（2/3以上表决权通过） | 南车北车合并：双方股东会高票通过，体现法律程序对重组合法性的保障 | 法律程序是重组启动的前提条件，案例中的高票通过反映了股东对协同效应的预期。 |
| 《上市公司重大资产重组管理办法》第11条：重大资产重组需报证监会核准 | 宝钢武钢合并：作为央企整合，除证监会外还需国资委审批、商务部反垄断审查 | 制度约束塑造了重组的时间成本与合规风险，案例中的"多部门审批"解释了重组周期较长的原因。 |
| 同一控制下企业合并采用"权益结合法"（账面价值计量） | 南车北车、宝钢武钢均采用此法，不确认商誉 | 会计方法选择直接决定了合并后财务报表的形态：无商誉→利润表无减值风险→便于分析真实经营改善。 |
| 非同一控制下企业合并采用"购买法"（公允价值计量），可能产生商誉 | 美的吸收合并小天鹅：需讨论是否存在商誉及减值测试安排 | 会计方法差异是案例比较的核心维度之一，直接影响"成功"的衡量标准（账面利润 vs 经济实质）。 |
| 美国Chapter 11的"债务人占有"（DIP）制度与自动中止条款 | 作为国际比较，反衬中国重组的行政主导特征；在结论中提出司法主导与行政主导的优劣比较 | 制度差异是解释"为何中国大型重组多以吸收合并而非破产重整完成"的理论基础。 |

### 4.3 各部分过渡段落要点

| 过渡位置 | 过渡段落核心句设计 |
| :--- | :--- |
| **第1章 → 第2章** | "要理解中国上市公司并购重组的实践逻辑，首先需在概念上厘清'企业重组'的边界，并在制度层面考察法律与监管为重组提供了怎样的规则框架。" |
| **第2章 → 第3章** | "制度法规为重组划定了'游戏规则'，而会计准则是记录重组经济后果的'记分牌'。本章将系统阐述不同重组方法对应的会计处理逻辑，为后续案例的财务分析奠定技术基础。" |
| **第3章 → 第4章** | "前述概念、制度与方法构成了分析并购重组的理论工具箱。本章将选取三个具有代表性的成功案例，将理论框架应用于具体情境，剖析重组动因、过程设计与会计处理细节。" |
| **第4章 → 第5章** | "案例叙述揭示了并购重组的'故事线'，但'成功'与否需要数据检验。本章运用财务比率分析与事件研究法，从会计报表与市场反应两个维度量化重组的价值创造效应。" |
| **第5章 → 第6章** | "财务分析表明，并购重组的成功并非自动实现，而是取决于战略契合度、整合执行力与会计政策选择的综合作用。基于前述发现，本章提出对准则制定、监管优化与企业实践的启示。" |
