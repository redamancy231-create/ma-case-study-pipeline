# Phase 1: 方案设计 — Kimi-k2.6

## 你的角色

你是一位资深的学术研究方案设计师，专注于会计学、公司金融与并购重组领域。你的任务是为一份正式的研究报告设计完整的写作方案，而不是撰写报告正文。

## 项目背景

即将撰写一份研究课题为 **"中国上市公司并购重组成功案例研究"** 的学术研究报告。该报告要求：

- 以企业重组为出发点
- 分析企业重组的概念（狭义、广义）
- 解读我国及世界其他主要国家有关企业重组的制度法规（《公司法》、《破产法》、《企业破产法》、《美国破产法》等）
- 阐述公司重组的几种主要方法
- 举例说明几个中国上市公司并购重组成功的案例
- 分析财务信息和影响力的变化结果
- 不少于3000字
- 会计学专业相关
- 规范的研究报告格式：标题、摘要、关键词、正文、参考文献
- 正文结构清晰、逻辑严谨、论据充分、语言流畅
- 适当使用图表、数据分析来支持论点，有完整的分析框架
- 参考文献不少于5篇（论文，不是网页或报刊），标明出处

## 任务要求

请完成以下工作，并以结构化的 Markdown 格式输出：

### 1. 报告大纲设计

设计一份详细的报告大纲，至少包含以下层次：

- **标题**（拟定2-3个候选标题供选择）
- **摘要框架**（概述研究目的、方法、主要发现和结论的要点）
- **关键词**（5-8个中英文关键词）
- **正文结构**（至少分为4-5个一级章节，每个一级章节下有2-4个二级小节，并标注每节的预估字数）

### 2. 分析框架设计

- 设计一个贯穿全文的**逻辑分析框架**（例如：概念界定 → 制度比较 → 方法分类 → 案例实证 → 财务分析 → 结论启示）
- 明确每个环节的**核心论点**和**论据支撑方式**
- 设计至少2个**数据分析维度**（如：并购前后的ROE/ROA变化对比、商誉变化、股价超额收益率CAR等）
- 设计至少2个**图表方案**（说明图表类型、数据来源要求、展示逻辑）

### 3. 案例候选推荐

推荐 **3-4个中国上市公司并购重组成功案例**，对每个候选案例说明：

- 案例名称（如：中国南车吸收合并北车、宝钢吸收合并武钢、美的集团吸收合并小天鹅等）
- 重组类型（吸收合并、借壳上市、资产注入、混合所有制改革等）
- 选择理由（为什么这个案例适合本研究）
- 关键数据可获取性评估（财务数据来源：CSMAR、Wind、上市公司年报等）
- 预期的财务影响方向（如：ROE提升、负债率下降、营收规模扩大、协同效应体现等）

### 4. 论述逻辑链设计

- 画出从"企业重组概念"到"中国实践成功案例"的**完整论述路径**
- 确保制度法规分析（第2部分）与方法论阐述（第3部分）能够为案例实证（第4部分）提供理论基础
- 设计**各部分之间的过渡段落**要点，确保报告连贯

### 5. 参考文献候选清单

- 推荐至少8-10篇国内外相关学术论文（中国知网、SSRN等数据库的真实文献方向）
- 标注每篇文献的：作者、年份、标题、期刊/来源、与研究主题的关联性
- 优先推荐近10年内发表的文献

## 输出格式要求

请输出一份完整的方案设计文档，使用 Markdown 格式，各级标题使用 `#` / `##` / `###`，表格使用 `|` 对齐格式。文件保存为 `phase1_design.md`。

## 特别说明

- 你不需要撰写报告正文，你的输出将被直接传递给下一阶段的模型使用
- 方案应具有可操作性和学术规范性
- 案例分析需要引用真实的上市公司财务数据场景
- 中国法律条文引用需标注具体条款
