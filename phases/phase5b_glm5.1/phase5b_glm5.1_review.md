## 1. 审查概述

本审查以211院校会计学本科毕业论文盲审标准，对终稿进行五个维度的学术规范复审：论文结构完整性与方案执行度（15%）、学术写作规范（25%）、论证逻辑严谨性（25%）、参考文献体系（20%）、术语统一与图表规范（15%）。审查依据终稿docx文件本体，对照phase1_design.md方案核验执行度，不对法律条款正确性或会计准则准确性作判断（归5-A）。审查方法包括：全文逐段阅读、参考文献逐一核对、图表编号顺序核验、术语一致性检索、论证链逻辑推演。

## 3. 第2部分 学术写作规范（权重25%）

### 3.1 摘要质量

**中文摘要核验：**

| 四要素 | 是否具备 | 说明 |
|---|---|---|
| 研究背景 | ✓ | "在资本市场深化改革……背景下" |
| 研究方法 | ✓ | "案例研究法、比较分析法、财务指标分析法和事件研究法" |
| 研究结论 | ✓ | 三点发现（战略协同基础、会计差异、CAR显著为正） |
| 研究贡献 | ⚠ 弱 | 末句"本文认为，并购重组的价值创造取决于交易后的资源整合、治理优化与风险控制"仅为一般性结论，未明确区分与既有研究的差异或本研究的独特贡献 |

**问题：**
- "本文"自指2次（"本文以……"、"本文认为……"），在可接受范围内
- 无口语化表达、无过度宣称

**英文Abstract核验：**

- 语法基本正确，无明显中式英语
- 时态使用合理：一般现在时为主（"examines", "evaluates", "suggest"），符合学术规范
- 被动语态使用恰当（"are examined", "is supported"）
- "those not under common control"表达可优化为"business combinations not under common control"或"non-common-control business combinations"

**关键词中英文对应：**

| 中文 | 英文 | 对应性 |
|---|---|---|
| 并购重组 | Mergers and Acquisitions | ✓ |
| 吸收合并 | Absorption Merger | ✓ |
| 协同效应 | Synergy Effect | ✓ |
| 财务绩效 | Financial Performance | ✓ |
| 商誉 | Goodwill | ✓ |
| 累计超额收益率 | Cumulative Abnormal Return | ✓ |

## 4. 第3部分 论证逻辑严谨性（权重25%）

### 4.1 整体论证链

终稿论证链为：**概念界定(2.1) → 理论基础(2.2) → 方法分类(2.3) → 制度环境(3.1-3.3) → 案例呈现(4.1-4.4) → 财务分析(5.1-5.5) → 经验启示(6.1-6.3)**

与要求链路"概念界定→理论基础→制度环境→案例呈现→财务分析→经验启示"一致。✓

**逐章核验：**

**第二章（概念/理论）：**
- 2.1概念界定为后续提供了"同一控制vs非同一控制""吸收合并vs新设合并"等核心分析工具 ✓
- 2.2理论部分列举了效率理论、代理理论、市场势力理论和交易费用理论四类
- **问题**：市场势力理论和交易费用理论在后文案例分析和财务分析中几乎未被调用，存在"概念列举但后文不再使用"的浪费。后文仅使用了效率理论（解释南车-北车、宝钢-武钢的规模效应）和代理理论（解释商誉减值风险），但未明确回扣到理论框架。

**第三章（制度）：**
- 3.1中国法律框架为案例的合规性分析提供了法律依据 ✓
- 3.2美国Chapter 11制度内容详细，但**后续三个案例均为中国上市公司，无美国破产重整案例**
- **问题**：3.2节美国制度内容在案例分析和财务分析中几乎未被引用，仅3.3节有简要对比，存在"背景陈列"嫌疑。建议在6.2启示部分回扣中美制度差异，或将美国制度部分精简为比较表（已有表2），减少纯陈述性内容。

**第四章（案例）：**
- 三案例覆盖同一控制吸收合并（南车-北车、宝钢-武钢）和非同一控制跨境并购（美的-库卡） ✓
- 4.4三案例比较涵盖了重组类型、会计分类、商誉影响和核心风险 ✓
- **问题**：**案例选择标准未在正文中明确说明**。Phase 1中有"选择理由"（政策标杆性、会计典型性、数据可得性等），但终稿正文未阐述"为何选这三个案例而非其他"。对211论文而言，案例选择标准的明确说明是研究方法论的必要组成部分。

- 5.1指标体系承接第二章理论框架（杜邦分解对应效率理论中的经营协同概念） ✓

**第六章（结论）：**
- 6.1三点结论由前文分析自然推导 ✓
- 6.2启示部分按"上市公司→会计准则与监管→投资者"三个维度展开，逻辑清晰 ✓

### 4.2 论据—论点匹配

**抽样审查3处"由数据/事实得出结论"的论证：**

**论证1**（5.3节，中国中车杜邦分解）：
> "净利率提升约0.85个百分点（来自规模化采购与海外订单溢价）"

- 数据支撑：销售净利率4.27%→5.12%，提升0.85个百分点 ✓
- 问题：括号内"来自规模化采购与海外订单溢价"为推测性归因，正文中未提供采购成本下降或海外订单溢价率的数据支撑。此为**因果推断过度**——数据仅表明净利率提升，但归因需要额外证据。
- 改前→改后：~~"净利率提升约0.85个百分点（来自规模化采购与海外订单溢价）"~~ → "净利率提升约0.85个百分点，可能归因于规模效应带来的采购成本优化及海外订单结构改善，但需更多细分数据验证"

**论证2**（5.2节，美的资产负债率上升）：
> "美的因现金收购库卡使资产负债率从2016年的59.34%上升至2017年的66.81%"

- 数据支撑：表5资产负债率59.34%→66.81% ✓
- 因果性：现金跨境收购确实会增加负债（银团贷款），因果推理合理 ✓
- 无问题

**论证3**（5.4节，CAR为正表明市场认可重组价值）：
> "三案例事件窗口CAR均显著为正，表明资本市场总体认可重组价值"

- 数据支撑：表7三个CAR均正且统计显著 ✓
- 问题：CAR为正可有多重解释（短期投机、政策预期、信息泄露等），直接等同于"认可重组价值"略嫌跳跃。此外，南车-北车[-5,+5]窗口CAR仅18.45%而[-30,+30]高达42.16%，差额约24个百分点，作者自己解释为"可能与'一带一路'相关政策预期共振有关"，但这恰恰说明CAR不仅反映"重组价值"本身。
- 改前→改后：~~"表明资本市场总体认可重组价值"~~ → "表明资本市场在公告窗口对重组持正面预期，但CAR亦可能包含政策预期、信息传递等多重因素，不宜简单等同于对重组价值的确认"
