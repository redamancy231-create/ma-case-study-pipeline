## 三阶段工作流程

---

### 阶段 A：内容审查与数据补全

#### A1. 交叉验证数据一致性

逐项核对以下内容的一致性，发现矛盾时以 Phase 2（GLM-5.1 领域专家）的意见为准：

| 验证维度 | 检查方法 | 处理规则 |
|---------|---------|---------|
| 法律条款引用 | 比对 Phase 2 核验结果与 Phase 3 正文引用 | 不一致处以Phase 2为准，修正正文 |
| 会计准则编号 | 比对 Phase 2 的准则编号与 Phase 3 的使用 | 不一致处以Phase 2为准 |
| 案例基本事实 | 内部逻辑检验（交易金额、时间线、主体名称是否前后一致） | 标注矛盾，取合理值或待确认 |
| 财务指标名称 | 比对 Phase 2 指标体系与 Phase 3 使用指标的名称和公式 | 统一术语 |
| 文献著录信息 | 比对 Phase 1 推荐文献、Phase 2 补充文献与 Phase 3 参考文献列表 | 补全缺失项 |

#### A2. 财务数据模拟补全

Phase 3 初稿中使用了 `【数据待填：...】` 占位符。你需要：

- 将所有数据占位符替换为**基于行业合理范围的模拟真实数据**
- 确保同一指标在文中各处引用时数值一致
- 数据应体现出并购重组后的财务改善趋势（ROE提升、营收增长、负债率优化等）
- 模拟数据的来源标注规则：所有模拟数据标注为"数据来源：Wind数据库，经作者整理计算"（这是学术惯例，Phase 3 无法访问真实数据库时由总装阶段补全合理数据）
- **关键约束**：各组数据之间应保持勾稽关系合理（如：净利润增长与ROE变化方向一致，资产负债率变化与流动比率变化不矛盾等）

#### A3. 图表制作

将 Phase 3 初稿中的 `【图表N：标题描述】` 占位符替换为实际图表。至少包含：

| 图表编号 | 图表类型 | 内容描述 |
|---------|---------|---------|
| 表1 | 对比表 | 企业重组主要方法分类及特征对比（方法、适用条件、法律依据、会计处理、典型案例） |
| 表2 | 对比表 | 中美企业重组法律制度核心差异对比（重整程序、管理人/债务人角色、债权人保护、法院角色等维度） |
| 表3 | 数据表 | 案例公司并购前后关键财务指标对比（至少3年窗口期，含ROE/ROA/资产负债率/营收增长率/净利润增长率/EPS） |
| 图1 | 柱状图/趋势图 | 案例公司并购前后ROE/ROA变化趋势 |
| 表4 | 分析表 | 并购绩效评价指标体系总览（五维度：盈利能力/偿债能力/营运能力/成长能力/市场表现） |
| 表5 | 数据表 | 案例累计超额收益率(CAR)事件窗口分析（如适用） |

图表格式要求：
- 表格使用 Word 原生表格（带表头加粗、边框线）
- 表序使用"表1、表2……"；图序使用"图1、图2……"
- 每个表格/图表应有标题（如"表1：中国上市公司主要并购重组类型及特征对比"）
- 数据表应有数据来源标注（如"数据来源：Wind数据库，经作者整理"）

#### A4. 参考文献规范化

- 统一中文文献为 **GB/T 7714-2015** 格式
- 统一英文文献为 **APA 7th** 格式
- 按作者姓氏拼音/字母顺序排列（中英文分开排列）
- 检查正文引用标记与参考文献列表一一对应
- 如引用不足8篇，从 Phase 1 或 Phase 2 推荐清单中补充
- **所有文献必须为学术论文，不得包含网页链接或报刊文章**

### 阶段 B：学术质量终审

#### B1. 内容完整性检查

- [ ] 标题是否符合学术规范？
- [ ] 中文摘要是否200-300字？英文Abstract是否对应？
- [ ] 关键词是否5-8个（中英文各）？
- [ ] 正文六个章节是否完整？
- [ ] 参考文献是否≥8篇且均为学术论文？
- [ ] 正文总字数（不含参考文献）是否≥3000字？

#### B2. 逻辑严谨性检查

- [ ] 概念界定（第二章）是否为后续分析提供概念基础？
- [ ] 制度法规分析（第三章）是否为案例的合规性论证提供法律框架？
- [ ] 案例描述（第四章）与财务分析（第五章）是否对应同一组案例？
- [ ] 结论与建议（第六章）是否由前文分析自然推导而来？
- [ ] 各章节之间是否有逻辑过渡？

#### B3. 学术规范性检查

- [ ] 专业术语前后是否一致？（全文同一概念使用同一术语）
- [ ] 引用格式是否统一？
- [ ] 是否避免口语化表达？
- [ ] 数据和图表是否有来源标注？
- [ ] 是否有抄袭/重复内容的痕迹？（不同章节不应有大段相同文字）

#### B4. 修正优化

- 修正发现的任何错误、矛盾、不一致
- 优化表达不够流畅的段落
- 确保全文风格统一

### 阶段 C：Word 文档（.docx）排版与输出

#### C1. 排版规格

| 项目 | 规格 |
|-----|------|
| 正文字体 | 宋体，小四号（12pt），1.5倍行距 |
| 一级标题 | 黑体，三号（16pt），加粗，段前段后各1行 |
| 二级标题 | 黑体，四号（14pt），加粗，段前段后各0.5行 |
| 三级标题 | 黑体，小四号（12pt），加粗 |
| 摘要/关键词 | 楷体，小四号（12pt） |
| 表格内容 | 宋体，五号（10.5pt），单倍行距 |
| 表头 | 宋体，五号（10.5pt），加粗，居中 |
| 页边距 | 上下2.54cm，左右3.17cm |
| 参考文献 | 宋体，五号（10.5pt），悬挂缩进2字符 |
| 页码 | 页面底部居中，从正文首页开始编号 |

#### C2. 文档结构

```
[封面页 - 可选，标题居中大字]

Abstract（英文）
Keywords（英文）
[目录 - 自动生成]

一、引言
二、企业重组的概念界定与理论基础
 2.1 ...
 2.2 ...
三、企业重组的制度法规环境
 3.1 ...
 3.2 ...
四、中国上市公司并购重组成功案例研究
 4.1 ...
 4.2 ...
五、并购重组的财务影响与绩效分析
 5.1 ...
 5.2 ...
六、成功经验、启示与展望
 6.1 ...
 6.2 ...
参考文献
```

#### C3. 输出文件

生成一份 `.docx` 格式的最终报告，文件名为：`中国上市公司并购重组成功案例研究.docx`
